# Project Templates Overview

This repository contains reusable frontend templates organized by **architecture pattern**.  
Each template follows a naming convention:

`<template-name>-<architecture-pattern>`

For example:  
- `shadcn-fsd` → uses **Shadcn UI** + **Feature-Sliced Design (FSD)**  
- `mantine-fsd` → uses **Mantine** + **FSD**  

Each template includes:  
- Preconfigured tech stack (framework, UI library, state management, dev tools)  
- Recommended FSD / architecture structure  
- Scripts for development, build, testing, and Storybook  
- Opinionated setup for faster project bootstrapping

> Goal: quickly start new projects with consistent architecture and tooling.
