# Обзор шаблонов проекта

В этом репозитории собраны переиспользуемые фронтенд-шаблоны, организованные по **архитектурным паттернам**.  
Каждый шаблон имеет систему именования:

`<имя-шаблона>-<архитектурный-паттерн>`


Примеры:  
- `shadcn-fsd` → использует **Shadcn UI** + **Feature-Sliced Design (FSD)**  
- `mantine-fsd` → использует **Mantine** + **FSD**  

Каждый шаблон включает:  
- Преднастроенный стек технологий (фреймворк, UI-библиотека, управление состоянием, инструменты разработки)  
- Рекомендованную структуру FSD / архитектуры  
- Скрипты для разработки, сборки, тестирования и Storybook  
- Опинионный setup для быстрого старта проектов

> Цель: быстро стартовать новые проекты с согласованной архитектурой и инструментами.
